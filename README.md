## About Blind 75

The Blind 75 is a list of 75 LeetCode questions compiled by a Facebook engineer that cover the most important concepts and patterns seen in coding interviews. The problems span various topics such as:

- Arrays
- Binary
- Dynamic Programming
- Graph
- Interval
- Linked List
- Matrix
- String
- Tree
- Heap
